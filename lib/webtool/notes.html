Webtool Release Notes

Webtool 0.8.1

Fixed errors and malfunctions

  • httpd.conf is removed. All config data for inets is given when the webserver is started with httpd:start2/1.
    (Own Id: OTP-4640)

Improvements and new features

  • Added shell script start_webtool and batch file start_webtool.bat for starting webtool including arbitrary application and browser.
    (Own Id: OTP-4748)
