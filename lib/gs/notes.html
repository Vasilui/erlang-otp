GS Release Notes

GS 1.5.2

Fixed errors and malfunctions

  • The GS font sizes have been adjusted so that they are more adequate i.e. smaller, closer to earlier releases.
    (Own Id: OTP-4653)
    (Aux Id: seq7823, seq7828, seq7922)

GS 1.5.1

Fixed errors and malfunctions

  • The communication between Erlang and the external Tcl/Tk process was improved.

    Compatibility problems with the Microsoft Windows version of Tcl/Tk 8 and menu handling was solved.

    The grid handing code was corrected to avoid intermittent crashes.

    gs:info() was changed not to crash when called with the wrong argument.
    (Own Id: OTP-4522)

GS 1.5.0

Fixed errors and malfunctions

  • gs:read(Canvas, {hit, {X,Y}}) crashed when executed over an image object.
    (Own Id: OTP-4167)
