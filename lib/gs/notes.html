GS Release Notes

GS 1.4.0

Improvements and new features

  • Code has been cleared up to get rid of compiler warnings.
    Own Id: OTP-3803
  • Some internal modules and processes in GS were prefixed with gtk. This prefix has been changed to gstk not to clash with modules that interface to the graphical toolkit GTk.
    Own Id: OTP-4065

Fixed errors and malfunctions

  • A reference to a function in toolbar_lib (in the Toolbar application) has been removed.
    Own Id: OTP-3967
