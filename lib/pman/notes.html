Pman Release Notes

Pman 2.4.1

Improvements and new features

  • cover_server (process belonging to the new tool Cover) has been added to the list of processes defined as system processes.
    Own Id: OTP-4008

  • The function defining the default settings file has been moved from GS to pman_osdepend.erl.
    Own Id: OTP-4022
    Aux Id: OTP-4021
  • Calls to gtk_name has been changed to gstk_name, see GS Release Notes.
    Own Id: OTP-4065

There are also release notes for older versions.
