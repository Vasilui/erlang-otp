Appmon Release Notes

Appmon 2.1.2

Known Problems

  • Appmon sometimes crashes if the application controller is heavily loaded, for example during a node startup.
    Own Id: OTP-3616

Fixed Errors and Malfunctions

  • Appmon assumed all connected nodes had the appmon code available. Now nodes where the appmon code is not available, for example embedded nodes, are ignored by Appmon.
    Own Id: OTP-3591
  • Child processes were shown in reversed order (compared to order of creation) when viewing an application tree in process mode.
    Own Id: OTP-4082

There are also release notes for older versions.
